# Retail Sales Forecast Dashboard

## 📌 Project Overview
This project is a Predictive Analytics Dashboard developed using Microsoft Power BI. The dashboard analyzes historical retail sales data to identify trends, monitor business performance, and forecast future sales.

The project uses time-series analysis and forecasting techniques to help businesses make data-driven decisions.

---

# 🎯 Objectives
- Analyze historical sales performance
- Forecast future sales trends
- Compare store-wise sales performance
- Monitor holiday vs non-holiday sales
- Create interactive and insightful dashboards

---

# 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Time-Series Forecasting

---

# 📂 Dataset Information
The dataset contains:
- Store ID
- Date
- Weekly Sales
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment

---

# 📊 Dashboard Features

## KPI Cards
- Total Sales
- Average Weekly Sales
- Maximum Weekly Sales

## Visualizations
- Weekly Sales Trend
- Sales Forecast Analysis
- Store-wise Sales Performance
- Holiday vs Non-Holiday Sales

## Interactive Features
- Store Filter
- Year Filter
- Dynamic Visual Interactions

---

# 📈 Forecasting
Power BI forecasting was used to predict future sales trends based on historical weekly sales data.

Forecast Settings:
- Forecast Length: 12 Weeks
- Confidence Interval: 95%
